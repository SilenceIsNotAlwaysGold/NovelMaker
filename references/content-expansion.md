# 内容扩充技巧

> 当章节字数不足3000字时，使用以下技巧自然扩充。
> 核心原则：扩充不是注水，每一段新增内容都必须服务于角色塑造、氛围营造或剧情推进。

## 技巧1：场景感官化（+300-500字）

把"他走进房间"扩展为沉浸式场景。

**扩充前：**
> 他走进了那间地下室。里面很暗。

**扩充后：**
> 铁门推开的声音在走廊里回荡了很久。地下室的空气又潮又闷，混着铁锈和不知道什么腐烂的气味。唯一的光源是头顶一盏摇晃的白炽灯，灯泡上爬满了飞蛾的尸体。他的鞋踩在水泥地上，每一步都踩出黏腻的回声。

**操作步骤：**
1. 挑选2-3种感官（视觉+嗅觉 或 听觉+触觉）
2. 用具体细节代替抽象描述
3. 让环境细节暗示情绪或危险

## 技巧2：动作分解（+200-400字）

关键动作不要一笔带过，分解成慢镜头。

**扩充前：**
> 他出剑了，很快，对方没反应过来就倒下了。

**扩充后：**
> 他的手指先动了。拇指顶住剑鞘口，其余四指在剑柄上找到了各自的位置。呼吸放慢。视线锁定对方的喉结——不，太高了——锁定锁骨下方三寸。
>
> 剑出鞘的声音只持续了半个呼吸。
>
> 对方低头看了一眼胸口，像是在确认那道红线是不是真的。然后膝盖弯了。

**适用场景：** 战斗高潮、关键操作、情感爆发点。不要滥用——日常动作（吃饭走路）不需要分解。

## 技巧3：对话加深（+300-600字）

对话不只是信息传递。加入潜台词、动作beats、对话中断。

**扩充前：**
> "你为什么帮我？"她问。
> "因为你需要帮助。"他说。

**扩充后：**
> "你为什么帮我？"她盯着他的眼睛，像是在找什么破绽。
>
> 他没有马上回答。手里的茶杯转了半圈。窗外的雨声填满了两人之间的沉默。
>
> "你真正想问的不是这个吧。"他把茶杯放下，"你想问的是，我帮你——要什么代价。"
>
> 她的手指收紧了。被说中了。

**对话扩充清单：**
- 加入动作beats（端杯子、转身、沉默）
- 角色不直接回答问题（绕弯子、反问）
- 加入环境音（雨声、脚步声）打破纯对话
- 身体语言泄露角色的真实想法

## 技巧4：内心外化（+200-400字）

不要"他很紧张"，而是用身体反应和行为来表现内心。

**扩充前：**
> 他很紧张。等待结果的时间太长了。

**扩充后：**
> 他已经把矿泉水的标签撕成了碎条。指甲缝里塞满了纸屑。门上方的电子钟显示15:47，他觉得至少过了两个小时，但其实才过了十一分钟。
>
> 走廊尽头有人推门的声音。他站起来——不是。只是个护士推着空推车走过。他又坐下了，椅子的塑料面发出一声抗议。

**对照表：**
| 情感 | 不要写 | 要写 |
|------|--------|------|
| 紧张 | 他很紧张 | 他在撕矿泉水标签 |
| 愤怒 | 他很愤怒 | 他的筷子折断了，他看了一眼，又拿了一双 |
| 悲伤 | 她很伤心 | 她把脸埋进围巾，呼出的热气让镜片起了雾 |
| 恐惧 | 他害怕了 | 他的后背贴着墙，能感觉到砖缝 |

## 技巧5：闪回/回忆（+200-500字）

在关键时刻插入短暂的记忆片段，加深角色厚度。

**使用规则：**
- 闪回不超过2段（太长会打断节奏）
- 必须与当前情节有直接关联
- 用斜体或时间标记与正文区分
- 闪回结束后必须有"回到现实"的转换句

**示例：**
> 那把刀的样子他见过。
>
> *七年前。厨房。母亲切蒜薹的时候，右手无名指上的旧疤正好对着灯光。他问过那道疤是怎么来的。母亲没有回答，只是换了一只手切菜。*
>
> 现在同样的刀握在陌生人手里。林远攥紧了拳头。

## 技巧6：次要互动（+200-400字）

在主线推进的间隙，加入角色之间的非剧情互动，让人物更立体。

**示例场景：**
- 赶路时的闲聊（展示角色关系和世界观）
- 吃饭时的细节（角色的饮食偏好揭示背景）
- 等待时的消遣（下棋、擦武器、看风景）
- 角色之间的小摩擦（不是冲突，是生活质感）

**关键：** 次要互动中也要埋信息。比如闲聊时不经意提到一个地名，后面会用到。

## 技巧7：环境压力（+200-300字）

用环境变化制造外部压力，增加紧迫感。

**示例：**
- 天黑了（必须在天亮前赶到）
- 下雨了（道路难走/视线受阻）
- 温度变化（越来越冷→说明方向走错了）
- 声音（远处传来的脚步声越来越近）

## 扩充禁忌

| 禁止 | 原因 |
|------|------|
| 大段无用风景描写 | 读者会跳过 |
| 重复已知信息 | 角色A向B复述读者已知的事 |
| 无冲突的日常流水账 | 即使是日常也需要小摩擦 |
| 强行加入新角色 | 只为凑字数出现的角色会让结构臃肿 |
| 过长的内心独白 | 超过3段就打断节奏 |

## 字数不足时的诊断流程

```
字数 < 3000？
  ↓
检查场景数量 → 太少？增加1个过渡场景（技巧6）
  ↓
检查对话密度 → 对话太干？用技巧3加深
  ↓
检查感官描写 → 太抽象？用技巧1具象化
  ↓
检查关键动作 → 写得太快？用技巧2分解
  ↓
检查情感展示 → 太直白？用技巧4外化
  ↓
仍不足？→ 在合理位置加一段闪回（技巧5）
```
