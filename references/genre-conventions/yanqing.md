# 言情 题材套路

## 核心要素
- **情感核心**：两人之间的吸引力、障碍、最终走向
- **化学反应**：对手戏要有张力（甜/虐/暧昧）
- **障碍设计**：不是误会制造机，要有真实的阻碍
- **HE/BE选择**：提前确定结局走向，铺垫要一致

## 经典结构
```
相遇（第一印象：讨厌/惊艳/误解）→ 被迫接触
→ 了解对方（发现闪光点）→ 暧昧期
→ 表白/确认关系 → 障碍出现（外部/内部）
→ 分离/冷战 → 重新理解 → 和好/升华
```

## 子类型区分

### 甜宠文
- 感情发展顺畅，男/女主全程宠对方
- 虐度低，甜度高，主打"被宠爱"的代入感
- 外部冲突为主（反派/家庭/事业），内部冲突弱
- 核心技巧：日常甜蜜细节的新鲜感（不能重复同样的甜）

### 虐恋/虐心文
- 感情之路坎坷，反复错过、误解、分离
- 虐度高，读者被虐到心疼→最终HE时的释放感
- 内部冲突为主（性格缺陷/过往创伤/立场对立）
- 核心技巧：虐要虐到点上（不是无理取闹），让读者心疼而非烦躁

### 先婚后爱
- 先有婚姻/契约关系，再逐渐产生感情
- 日久生情的慢热过程是最大看点
- 生活细节>戏剧冲突
- 核心技巧：相处中的细微变化（从冷淡→习惯→在意→离不开）

### 年代文/古言
- 时代背景对感情的压制和塑造
- 社会规则是最大的障碍（门当户对/父母之命/社会舆论）
- 语言风格要匹配时代
- 核心技巧：在限制中寻找表达爱意的方式更动人

### 破镜重圆
- 曾经相爱→因故分开→重逢→纠结→复合
- 核心张力：旧情+怨恨+不甘心的复杂情绪
- 必须揭示当初分开的真相
- 核心技巧：两人独处时的克制与心理活动

## 经典感情线模板

### 欢喜冤家
```
互看不顺眼 → 被迫合作 → 发现对方优点
→ 吵着吵着就亲了 → 嘴上说讨厌身体很诚实
```
**关键节点**：
- 吵架中不经意流露的关心
- "我才不是担心你"式的口是心非
- 从"看不顺眼"到"看不到就心烦"的转变

### 先婚后爱
```
契约婚姻 → 冷漠共处 → 日久生情
→ 一方先动心 → 另一方觉察 → 双向奔赴
```
**关键节点**：
- 第一次为对方做出改变（比如调闹钟做早餐）
- 发现对方不在时的失落感
- 被外人问"你们关系很好"时的反应差异

### 暗恋成真
```
单方面暗恋 → 默默付出 → 被发现
→ 感动/心动 → 双向奔赴
```
**关键节点**：
- 暗恋者以为自己藏得很好（其实所有人都看出来了）
- 被暗恋者对"付出"习以为常后突然失去→才意识到重要性
- 暗恋者准备放弃时的告白

### 破镜重圆
```
曾经相爱 → 误会/变故分开 → 重逢
→ 纠结 → 解开心结 → 复合
```
**关键节点**：
- 重逢时的强装冷漠（内心翻涌）
- 发现当初的真相（原来你也在承受）
- "我恨你"说出口→眼泪先掉下来

## 甜虐节奏控制

### 节奏公式
```
甜 → 甜 → 小虐 → 甜（补偿）→ 大虐 → 长甜
```

**原则**：
- 虐完必须补甜，不能连续虐超过3章（读者会弃书）
- 甜虐比例建议：甜宠文 8:2，虐恋 5:5，先婚后爱 7:3
- 虐的强度要递进：小虐→中虐→大虐→终极虐→大甜结局

### 不同阶段的甜虐设计
| 阶段 | 甜 | 虐 |
|------|-----|-----|
| 暧昧期 | 偶然接触、意外心动 | 误解对方的意图 |
| 确认期 | 告白、第一次牵手 | 外部阻力出现 |
| 热恋期 | 日常相处、甜蜜互动 | 第三者/事业冲突 |
| 危机期 | 回忆过去的甜蜜 | 分离/冷战/背叛（误） |
| 回归期 | 和好的心酸与释然 | 为对方做出牺牲 |

## 化学反应写法

### 暧昧的十个台阶
1. **目光追随**：不自觉地找对方在哪
2. **无意识靠近**：坐在一起时缩短距离
3. **偶然身体接触**：碰到手→触电→收回
4. **记住细节**：对方的习惯/喜好了如指掌
5. **吃醋**：看到对方和别人亲近→心里不舒服
6. **心跳加速**：对方靠近时的生理反应
7. **专属称呼/行为**：只对你一个人这样
8. **保护欲**：下意识挡在前面/把安全的位置给你
9. **占有欲**：介绍时说"我的朋友"→"我的人"
10. **告白/确认**：从暧昧到明确

### 写化学反应的技巧
- **潜台词>直说**：不说"我喜欢你"，说"你什么时候走？……我没有等你的意思"
- **身体语言>语言告白**：拨开她脸上的头发比说"你很美"更有张力
- **克制比放纵更动人**：明明想牵手却把手插进口袋
- **第三视角旁证**：让旁人说"你对他不一样"
- **拉扯感**：前进一步→退后两步→再前进三步

## 对话是言情的灵魂

### 不同阶段的对话特征
| 阶段 | 对话特点 | 示例风格 |
|------|----------|----------|
| 初识 | 客气/带刺/试探 | "哦，你就是那个……" |
| 暧昧 | 潜台词多/欲言又止 | "你不用来的。""我路过。" |
| 吵架 | 每句都带情绪/翻旧账 | "你从来都是这样！""我什么样？" |
| 甜蜜 | 日常废话/专属玩笑 | "今天想吃什么？""你做什么我吃什么。" |
| 虐心 | 话少/克制/说反话 | "我没事。""那就好。"（两人都在忍眼泪）|

### 对话禁忌
1. **误会制造机**：不能靠角色不说清楚来制造冲突
2. **过度解释**：角色不需要把内心独白说出来
3. **全程高甜**：没有冲突的对话没有张力
4. **智商下线**：为了剧情需要让角色说蠢话

## 障碍设计

### 好的障碍 vs 烂的障碍
| 好的障碍 | 烂的障碍 |
|----------|----------|
| 阶层差距（真实存在的壁垒）| 靠不说清楚制造的误会 |
| 立场冲突（各自有正当的理由）| 突然冒出的前女友/前男友 |
| 性格缺陷（角色自身的问题）| 反派毫无理由的从中作梗 |
| 过往创伤（心理上的真实障碍）| 失忆（除非处理得好）|
| 家族/时代压力（外部真实困境）| 血缘关系的狗血反转 |

### 障碍的节奏
- 小障碍（1-2章解决）→ 中等障碍（3-5章）→ 大障碍（半卷）→ 终极障碍（最后1-2卷）
- 每个障碍解决后感情要有实质性进展
- 不能反复用同一类型的障碍（读者会烦）

## 角色原型

### 男主原型
| 类型 | 特征 | 适合搭配 | 注意 |
|------|------|----------|------|
| 霸道总裁 | 外冷内热/控制欲/全能 | 独立女主/元气少女 | 控制欲别变成PUA |
| 温润学长 | 温柔/体贴/隐藏深情 | 倔强女主/受伤女主 | 不要太完美失真 |
| 痞帅竹马 | 嘴贱心善/暗恋已久 | 傲娇女主/学霸女主 | 痞要有度不是渣 |
| 病娇/黑化 | 偏执/占有/深情到扭曲 | 善良女主/温柔女主 | 要有合理成因 |
| 忠犬型 | 始终守护/无条件支持 | 事业女主/女强 | 不要变成舔狗 |

### 女主原型
| 类型 | 特征 | 适合搭配 | 注意 |
|------|------|----------|------|
| 独立女强人 | 事业有成/不依附男人 | 霸总/温润型 | 不要"嘴上女强"实际恋爱脑 |
| 元气少女 | 阳光/治愈/感染力强 | 冷面男主/病娇 | 元气不等于脑残 |
| 傲娇/毒舌 | 嘴硬心软/高冷外表 | 痞帅/温柔型 | 傲娇有度不是作精 |
| 温柔坚韧 | 表面柔弱/内心强大 | 霸道型/痞帅 | 不要变成圣母 |
| 复仇/重生 | 带着恨意或记忆归来 | 忠犬型/同盟型 | 复仇要有底线 |

## 避坑指南

### 严重问题（必须避免）
1. **误会推动全篇**：不能靠角色不沟通来撑100章
2. **人设崩塌**：前面独立后面突然变恋爱脑
3. **感情线没铺垫**：突然就爱上了，没有心动过程
4. **三观有问题**：美化PUA/家暴/跟踪/控制
5. **女配工具人**：反派女配只为衬托女主而存在

### 常见问题（注意规避）
6. **节奏失衡**：暧昧期30章、在一起后2章就结局
7. **甜腻过头**：全程高甜没有冲突，读起来像白开水
8. **虐过头**：虐到读者弃书，没有喘息空间
9. **配角比主角好磕**：副CP太出彩抢戏
10. **床戏替代感情**：用身体接触代替心灵沟通
11. **同质化**：所有男主都是霸道总裁，所有女主都是灰姑娘
12. **结局太仓促**：矛盾解决太快，HE来得太敷衍

### 写作建议
- 写言情前先想清楚：他们为什么爱上对方？（具体的点，不是"因为帅/美"）
- 最好的甜蜜场景是日常小事：一起买菜、帮忙吹头发、记住对方的咖啡口味
- 虐之前先让读者爱上这对CP，否则虐了也不心疼
- 感情发展的每一步都要有事件驱动，不能"忽然有一天就心动了"
- 写完每章问自己：这一章感情线有没有推进？哪怕一厘米也好
