# 历史/架空 题材套路

## 核心要素
- **时代还原**：语言、礼仪、制度、生活细节要有时代感
- **权谋博弈**：朝堂/后宫/军事是核心戏剧场
- **人在历史中**：大时代下个人的命运和选择
- **架空自由度**：架空历史可以改变走向，但内部逻辑要自洽

## 经典结构
```
入局（身份确立/穿越/出仕）→ 初试锋芒（小规模博弈）
→ 站队/选择阵营 → 权力上升 → 遭遇政敌/大危机
→ 失势/低谷 → 反击/翻盘 → 权力巅峰/功成身退
```

## 子类型区分

### 朝堂权谋
- 以官场博弈为核心
- 重策略/人心/信息战
- 核心看点：布局→执行→翻盘
- 需要扎实的政治制度知识

### 后宫争斗
- 后宫为主战场，妃嫔争宠
- 女性视角，重人心和手段
- 核心看点：结盟/背叛/上位
- 注意：不要把女性角色都写成阴险毒妇

### 军事征战
- 以战争/军事为主线
- 战术/战略/兵种/后勤
- 核心看点：以少胜多/奇计取胜
- 需要基本的军事常识

### 穿越/重生
- 现代人穿越到古代/架空世界
- 用现代知识改变历史
- 核心看点：先知优势+文化冲突
- 注意：不能用现代三观强行改造古代社会

### 架空王朝
- 完全虚构的王朝/国家
- 自由度最高，不受真实历史约束
- 可以融合多个朝代的元素
- 注意：架空不等于没有规则

## 常用冲突模板

| 冲突类型 | 描述 | 写法要点 |
|----------|------|----------|
| 夺嫡之争 | 几个皇子/继承人争夺皇位 | 每个势力要有合理的支持基础 |
| 权臣vs皇帝 | 功高震主/专权vs收权 | 双方都有正当性 |
| 新旧势力 | 改革派vs保守派 | 改革不能全对，保守不能全错 |
| 外敌入侵 | 边境战争/异族压境 | 不要把敌方写成野蛮人 |
| 党争 | 朝堂派系斗争 | 利益>理念，但也不全是 |
| 农民起义 | 底层反抗/推翻暴政 | 起义的组织和困难要真实 |
| 后宫博弈 | 妃嫔间的权力角逐 | 不只是争宠，是政治联盟 |
| 谏言冒死 | 正直大臣犯颜直谏 | 展示勇气和代价 |

## 权谋写作技巧

### 博弈的五个层次
```
第一层：信息战（知道别人不知道的事）
第二层：人心术（判断对方会怎么反应）
第三层：借势（利用形势/制度/规则）
第四层：造势（主动创造有利局面）
第五层：布局（多步棋连环）
```

### 权谋场景写法
| 场景 | 关键 | 忌讳 |
|------|------|------|
| 朝堂议事 | 每个发言都有立场，沉默也是态度 | 角色排队发言像开会 |
| 密室谈判 | 试探→底牌→交易，台面下的交换 | 直接亮底牌没有过程 |
| 设局陷害 | 多步骤执行，每步都有备选方案 | 一步到位太简单 |
| 翻盘反击 | 利用对手的弱点/过度自信 | 主角靠运气翻盘 |
| 识人用人 | 慧眼识才/驭人有术 | 所有人才都主动来投 |

### 权力的代价
- 每次权力上升都要付出代价（牺牲关系/原则/时间）
- 权力越大越孤独——这是永恒的主题
- 展示"不得不做的恶"比"主角全善"更有深度

## 时代感营造

### 语言层面
| 类型 | 可以用 | 不能用 |
|------|--------|--------|
| 称呼 | 臣/妾/公子/大人 | 你好/OK/没问题 |
| 敬语 | 不敢/在下/阁下 | 能不能/行不行 |
| 日常用语 | 用膳/安寝/传召 | 吃饭/睡觉/叫他来 |
| 情感表达 | 心悦/倾慕/惦念 | 喜欢你/爱你 |

**注意**：不需要完全用文言文，但关键场合（朝堂/诏书/对长辈）语言要有古感。日常对话可以偏白话。

### 生活细节
| 细节 | 需要注意 |
|------|----------|
| 饮食 | 什么朝代有什么食物（辣椒明末才传入） |
| 服饰 | 不同身份的服制/颜色限制 |
| 交通 | 马车/轿子/骑马的身份差异 |
| 礼仪 | 跪拜/作揖/请安的场合和对象 |
| 计时 | 时辰制/更鼓/日晷 |
| 住所 | 府邸/宫殿/民居的规制 |

### 架空设定的自由度
- 可以混合多个朝代的制度（唐的开放+宋的文治+明的锦衣卫）
- 可以改变性别规则（女帝/女官）
- 可以加入虚构的制度（但要有内在逻辑）
- 不能出现明显的时代错乱（蒸汽机+封建制度需要解释）

## 角色原型

### 主角原型
| 类型 | 特征 | 适合开局 |
|------|------|----------|
| 权谋之子 | 城府深/善于博弈 | 夺嫡/入朝 |
| 忠义之臣 | 正直/有原则/为社稷 | 犯颜直谏/守边 |
| 穿越者 | 先知优势/文化冲突 | 穿越/重生 |
| 草根崛起 | 寒门出身/科举上位 | 赴京赶考/投军 |
| 末代皇族 | 国破家亡/复国或隐居 | 亡国/逃亡 |
| 后宫女主 | 智慧+手腕+底线 | 入宫/选秀 |

### 反派/对手设计
- **权臣型**：有能力有野心，不完全是坏人
- **世家型**：维护家族利益，阻碍改革
- **外敌型**：有自己的文明和正当诉求
- **帝王型**：猜忌心重但也有苦衷

## 历史文的节奏

### 单卷节奏模板
```
第1-2章：入局+确立身份和处境
第3-4章：初次博弈（小规模试探）★
第5-6章：结盟+情报收集
第7章：第一次危机（被人算计）
第8-9章：反击+翻盘★★
第10章：地位上升+新的对手出现
第11-12章：更大的博弈（朝堂/战场）
第13章：遭遇重大挫折（低谷）
第14章：绝境中的突破★★★
第15章：阶段性胜利+更大的格局揭示★★★★
```

### 战争场景的节奏
- 战前（筹备/侦查/布阵）> 战中（厮杀描写）> 战后（收拾/代价）
- 战中字数不超过40%，重在战术和人心
- 大战前要有"安静的一刻"（战前夜话/遗书/祈祷）

## 避坑指南

### 严重问题（必须避免）
1. **现代思维强加古代**：用现代女权/民主概念改造古代社会需要铺垫
2. **权谋太幼稚**：朝堂大臣都像小学生一样好骗
3. **历史常识错误**：张冠李戴的制度/物品/事件
4. **穿越万能**：穿越者靠记忆轻松碾压所有人
5. **后宫只有争斗**：把所有女性都写成工于心计

### 常见问题（注意规避）
6. **信息来源不合理**：主角总能及时得到关键情报
7. **古人太傻**：为了衬托主角让古人智商下线
8. **战争儿戏**：几百人打来打去就决定天下
9. **感情线突兀**：朝堂上杀伐果断，一见美人就智商归零
10. **阶层流动太容易**：寒门到宰相不是一两年的事
11. **礼仪错误**：见皇帝的规矩/后宫的等级混乱
12. **太过黑暗**：所有人都在算计，没有一个好人

### 写作建议
- 写权谋前先想清楚：各方势力的利益诉求是什么？
- 每个角色做决策时要问：他的信息量是多少？（不能全知）
- 战争不是只有打仗：后勤、士气、天气、地形都影响战局
- 架空历史的好处是自由，坏处是容易自相矛盾——设定后要坚持
- 最好的历史文让读者感受到"时代洪流中个人的渺小与挣扎"
