# 都市 题材套路

## 核心要素
- **贴近现实**：场景、职业、社交关系要有代入感
- **身份反差**：主角表面普通，暗藏实力/背景
- **社会阶层**：阶层碰撞、底层逆袭是核心戏剧冲突
- **爽点来源**：装逼打脸、商战智斗、情感纠葛

## 经典结构
```
底层开局 → 遭受欺压 → 暴露实力/获得机遇 → 初步逆袭
→ 进入上层圈子 → 商场/官场/江湖博弈 → 更大的对手
→ 终极对决 → 站在顶端
```

## 子类型区分

### 都市异能
- 现代世界中存在超自然力量（异能/修炼/灵气复苏）
- 主角在正常社会和异能圈之间切换
- 核心冲突：暗中保护身边人+对抗异能反派
- 注意：异能要有限制，不能变成超人

### 商战都市
- 以商业竞争为主线，智斗>武斗
- 主角靠头脑+先知（重生/系统）赚钱
- 核心爽点：商业逆转、收购打脸、投资暴富
- 注意：商业逻辑要经得起推敲

### 医武都市
- 主角身兼医术和武力
- 医术用于结交权贵/救人，武力用于护身
- 核心套路：治好不治之症→获得人脉→碾压看不起自己的人
- 注意：医术描写要有最基本的合理性

### 重生/回到过去
- 带着未来记忆回到过去
- 核心爽点：预知未来+避免遗憾+改变命运
- 先知优势要逐渐失效（世界线偏移）
- 注意：不能全靠记忆碾压，角色本身要有成长

## 常用冲突模板

| 冲突类型 | 描述 | 写法要点 |
|----------|------|----------|
| 职场打压 | 上司/同事欺负→能力碾压 | 先展示主角的专业能力，再安排不公平待遇 |
| 相亲打脸 | 被嫌弃→暴露真实身份 | 嫌弃场景要写得让读者气愤，反转才爽 |
| 商战 | 公司竞争/收购/背叛 | 商战逻辑要清晰，不能靠主角一拍脑袋就赢 |
| 阶层冲突 | 富二代欺人→主角反杀 | 富二代要有靠山，不能一打就倒 |
| 家族回归 | 被赶出家族→实力回归 | 回归时机选在家族最需要他的时候 |
| 感情纠葛 | 前任/情敌/误会 | 感情线不能喧宾夺主，服务于主线 |
| 圈层突破 | 从底层进入上流社会 | 展示文化冲突和阶层差异的细节 |
| 身份暴露 | 隐藏身份被发现 | 暴露要在最有戏剧性的时刻 |

## 都市场景写作要点

### 场景真实感
| 场景 | 必须包含的细节 | 避免 |
|------|----------------|------|
| 公司/职场 | 工位布局、打卡、同事八卦、绩效压力 | 空洞地写"来到公司" |
| 饭局/酒桌 | 座次安排、敬酒规矩、暗中较劲 | 只写"大家喝酒聊天" |
| 医院 | 挂号排队、科室走廊、消毒水味道 | 主角随便进手术室 |
| 拍卖会/商会 | 入场资格、竞拍规则、暗中串通 | 主角想来就来 |
| 高档餐厅/会所 | 预约制度、着装要求、服务细节 | 穿着拖鞋就进去了 |
| 街头/夜市 | 烟火气、摊贩吆喝、路人反应 | 所有场景都高大上 |

### 金钱描写规范
- 数字要有现实感：不要动辄就上亿（除非商战大佬）
- 赚钱方式要合理：解释清楚钱从哪来
- 消费场景要有对比：普通人的反应衬托金额大小
- 避免"数字爽"：不要把都市文变成数字膨胀游戏

## 装逼打脸的正确写法

### 黄金公式
```
铺垫（被看不起/被欺负/被误解）
→ 蓄势（配角嘲笑/旁人不看好/对手得意）
→ 反转（暴露实力/身份/关系）
→ 打脸（对方震惊/后悔/道歉）
→ 余韵（旁观者议论/传播/后续影响）
```

### 关键原则
1. **被迫暴露比主动炫耀更爽**：主角不要主动装逼
2. **旁观者反应很重要**：围观群众的震惊/议论放大爽感
3. **打脸对象要有分量**：打无名小卒不爽，打有头有脸的人才爽
4. **后续效应**：打脸后要有涟漪效应（传出去/影响后续剧情）
5. **频率控制**：不能每章都打脸，会审美疲劳

### 不同层级的打脸设计
| 层级 | 对象 | 场景 | 频率 |
|------|------|------|------|
| 小打脸 | 路人/小喽啰 | 日常场景中的小冲突 | 每3-5章 |
| 中打脸 | 有背景的对手 | 宴会/比赛/商业场合 | 每卷2-3次 |
| 大打脸 | 重量级人物 | 重要场合+多方见证 | 每卷1次 |
| 终极打脸 | 大反派/家族 | 全书关键节点 | 全书2-3次 |

## 角色原型

### 主角原型
| 类型 | 特征 | 开局方式 | 注意 |
|------|------|----------|------|
| 隐藏大佬 | 退隐回归/扮猪吃虎 | 退伍/下山/回国 | 暴露节奏要控制 |
| 底层逆袭 | 白手起家，靠能力上位 | 被裁/被分手/被看不起 | 逆袭过程要有挫折 |
| 重生知未来 | 带着记忆重活一次 | 回到关键时间节点 | 先知优势要逐渐减弱 |
| 赘婿觉醒 | 入赘受辱→暴露身份 | 在女方家族受气 | 不要写成对女性的报复 |

### 都市反派设计
- **不能只是"坏"**：要有自洽的行为逻辑
- **背景要有层次**：小反派后面有中反派，中反派后面有大反派
- **反派也有人性**：偶尔展示反派的另一面（对家人好等）
- **反派的下场要多样**：不是每个都死/惨，可以被收服/转变

## 爽点分布

### 都市文节奏特点
- 比玄幻慢，重在对话和智斗
- 爽点来源更日常化（比赛赢了/赚到钱/被认可）
- 情感线占比更大（恋爱/友情/亲情）

### 单卷节奏模板（15章示例）
```
第1章：遭遇不公平/被欺负→引发读者共情
第2-3章：隐忍+暗中准备
第4章：小场合初露锋芒★
第5-6章：进入新圈层+遇到更大的挑战
第7章：饭局/酒桌上的暗中较劲★★
第8-9章：商战/职场对决
第10章：遭遇挫折/背叛（低谷）
第11-12章：调整+获得新的支持/资源
第13章：关键场合的大逆转★★★
第14章：打脸+收获★★★★
第15章：更大的格局揭示+下卷伏笔★★★★★
```

## 避坑指南

### 严重问题（必须避免）
1. **现实逻辑崩塌**：赚钱方式不合理、社会规则不对
2. **女性角色物化**：女角色不能只是主角的战利品
3. **价值观偏差**：不要传递"有钱就是爷"的单一价值观
4. **暴力解决一切**：都市不是法外之地，打人要有后果
5. **脸谱化**：富人全坏/穷人全好这种二元对立

### 常见问题（注意规避）
6. **社交关系不真实**：现实中人际关系更复杂
7. **赚钱速度太快**：白手起家到亿万富翁不能只用几章
8. **配角智商下线**：为了衬托主角让所有人变蠢
9. **打脸太频繁**：每章都打脸会麻木
10. **情感线过多**：一个主角身边不要超过3个暧昧对象
11. **职业描写不专业**：写医生/律师/金融要做基本功课
12. **阶层跨越太轻松**：真正的阶层壁垒不是一个人能轻松打破的

### 写作建议
- 多观察现实生活中的社交细节和阶层差异
- 每个商业决策都要有逻辑链，不能"灵机一动"
- 都市文的对话比叙述更重要，对话质量决定小说质量
- 适当加入社会热点话题增加代入感（但不要太敏感）
