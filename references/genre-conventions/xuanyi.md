# 悬疑/推理 题材套路

## 核心要素
- **谜题驱动**：每卷一个核心谜题，全书一个终极真相
- **线索管理**：真线索+假线索+误导，节奏释放
- **逻辑闭环**：所有谜底必须有前文线索支撑，禁止机械降神
- **氛围营造**：不安感、怀疑感、真相临近的紧迫感

## 经典结构
```
事件发生（谁死了/什么消失了）→ 初步调查
→ 嫌疑人出现（每人都有动机）→ 线索收集
→ 第一次误判 → 新证据推翻
→ 真相接近（侦探也陷入危险）→ 最终揭示
→ 反转（真凶出人意料但合乎逻辑）
```

## 子类型区分

### 本格推理
- 以逻辑推理为核心，读者可以和侦探同步推理
- 重密室/不在场证明/物理诡计
- 公平性原则：所有破案线索必须提前展示
- 注意：诡计设计要经得起逻辑推敲

### 社会派推理
- 犯罪背后是社会问题（阶层/歧视/制度缺陷）
- 重"为什么杀人"而非"怎么杀人"
- 犯人往往有让人同情的动机
- 适合探讨人性与社会议题

### 悬疑惊悚
- 氛围>推理，恐惧感是核心体验
- 可以有超自然元素（或最终科学解释）
- 重节奏控制和信息释放
- POV 切换制造紧张感

### 犯罪心理
- 以犯罪者心理为切入点
- 侦探vs犯罪者的心理博弈
- 犯罪者的行为有心理学依据
- 适合写连环案件

## 线索释放节奏

| 阶段 | 释放内容 | 读者状态 | 写法要点 |
|------|----------|----------|----------|
| 前25% | 核心谜题+3-4个嫌疑人 | 好奇 | 快速建立悬念，每个嫌疑人都有动机 |
| 25-50% | 部分线索+误导信息 | 开始推理 | 给出真线索的同时埋入红鲱鱼 |
| 50-75% | 关键线索+第一次反转 | 以为知道答案 | 推翻读者的第一判断 |
| 75-90% | 真相碎片+侦探陷入危险 | 紧张 | 信息加速释放，节奏收紧 |
| 90-100% | 真相揭示+终极反转 | 恍然大悟 | 所有线索串联，逻辑闭环 |

## 线索管理系统

### 线索分类
| 类型 | 作用 | 数量建议 | 说明 |
|------|------|----------|------|
| 核心线索 | 直接指向真相 | 3-5条 | 分散在不同章节，早期给但读者不会注意 |
| 辅助线索 | 排除嫌疑人/缩小范围 | 5-8条 | 帮助推进排除法 |
| 红鲱鱼 | 误导读者 | 2-3条 | 看起来指向某人但最终被推翻 |
| 隐藏线索 | 回头才能发现的伏笔 | 2-4条 | 嵌入日常描写中，初读不会注意 |

### 线索埋设技巧
1. **藏在日常中**：重要线索混在无关细节里，读者初读时不会注意
2. **分离呈现**：将一条完整线索拆成2-3部分，分别出现
3. **注意力转移**：在给出关键线索的同时发生更吸引眼球的事件
4. **角色口中**：让某角色不经意提到的话成为关键线索
5. **物理证据**：物品的位置/状态/痕迹是最可靠的线索载体

### 红鲱鱼设计
```
建立嫌疑人A的动机（看起来最可疑）
→ 给出指向A的证据（让读者相信是A）
→ 中期出现新证据否定A（读者困惑）
→ 揭示A有隐藏秘密但与案件无关
→ 真凶是之前不太引人注目的角色
```

## 嫌疑人设计

### 黄金配置（5-7人）
| 角色 | 嫌疑程度 | 实际身份 | 设计目的 |
|------|----------|----------|----------|
| 最可疑者 | ★★★★★ | 无辜（有自己的秘密）| 吸引读者注意力 |
| 次可疑者 | ★★★★ | 无辜（但行为可疑）| 备选嫌疑人 |
| 有动机者 | ★★★ | 无辜（动机真实但未行动）| 增加复杂度 |
| 目击者/帮手 | ★★ | 可能是共犯 | 提供线索的人 |
| 不起眼者 | ★ | 真凶 | 出现但不引人注目 |
| 受害者关系人 | ★★★ | 无辜（悲伤/愤怒）| 提供背景信息 |

### 嫌疑人排除节奏
- 每3-5章排除1个嫌疑人
- 排除方式要有新信息支撑，不能无端洗白
- 被排除的角色可能仍有其他秘密（增加复杂度）
- 真凶在被怀疑时要有合理的脱嫌理由

## 反转设计

### 反转层级
| 级别 | 类型 | 使用频率 | 示例 |
|------|------|----------|------|
| 小反转 | 新信息推翻旧认知 | 每3-5章 | 不在场证明被打破 |
| 中反转 | 嫌疑人身份/关系变化 | 每卷2-3次 | 发现嫌疑人是受害者亲属 |
| 大反转 | 案件本质改变 | 每卷1次 | 自杀被包装成谋杀 |
| 终极反转 | 全书级真相颠覆 | 全书1-2次 | 侦探就是凶手的帮凶 |

### 反转的基本原则
1. **出乎意料但合乎逻辑**：反转后回顾前文要能找到线索
2. **不能为反转而反转**：每次反转要推进剧情或深化主题
3. **反转不能太密集**：连续反转会让读者疲劳
4. **最终真相要比所有假设更合理**：读者的"恍然大悟"来自逻辑满足感

## 叙述性诡计

### 常用手法
| 手法 | 原理 | 注意事项 |
|------|------|----------|
| 身份误导 | 利用代称/绰号让读者误以为是另一人 | 回看时要能发现端倪 |
| 时间错位 | 不同视角的叙述并非同一时间 | 要有时间线索（天气/日期暗示） |
| 视角盲区 | 第一人称叙述者隐瞒信息 | 不能让叙述者说谎（只能不提） |
| 双重含义 | 某句话/物品有两层理解 | 两层含义都要自洽 |
| 可靠性欺骗 | 叙述者记忆不准确/有精神问题 | 提前暗示叙述者不可靠 |

## 氛围营造技巧

### 五感营造不安
| 感官 | 技巧 | 示例 |
|------|------|------|
| 视觉 | 昏暗/闪烁/阴影 | 走廊尽头的灯忽明忽灭 |
| 听觉 | 异响/寂静/不协调的声音 | 空房间里传出脚步声 |
| 嗅觉 | 腐败/消毒水/不明气味 | 甜腻的花香下面藏着什么 |
| 触觉 | 温度变化/质感异常 | 门把手是湿的 |
| 第六感 | 直觉不安/被注视感 | 总觉得有人在看自己 |

### 信息控制节奏
- **紧**：短句、快切、多对话 → 追逐/发现线索/面对危险
- **松**：长句、环境描写、内心独白 → 分析/推理/回顾
- **紧松交替**：每2-3章一个节奏循环

## 公平推理的底线

### 必须遵守
1. 所有推理所需的线索必须在揭示前给读者看过
2. 侦探不能用读者不知道的信息破案
3. 不能靠巧合/超自然解决谜题（除非提前建立规则）
4. 犯人的动机必须在前文有暗示
5. 证据链不能有逻辑断裂

### 可以使用
- 将线索藏在不显眼处（公平但不明显）
- 利用读者的思维定势误导
- 给出线索但延迟其意义（后来才知道重要性）

## 避坑指南

### 严重问题（必须避免）
1. **机械降神**：突然冒出之前没有任何铺垫的证据
2. **逻辑漏洞**：推理链条有明显不合理处
3. **真凶无端**：凶手的动机太牵强或前文完全没暗示
4. **虎头蛇尾**：铺垫很好但揭秘令人失望
5. **过度依赖巧合**：关键证据靠主角偶然发现

### 常见问题（注意规避）
6. **侦探全知**：主角不应该什么都猜得到
7. **推理过程枯燥**：纯逻辑分析没有情感和冲突
8. **嫌疑人性格单薄**：每个嫌疑人要有自己的故事
9. **线索堆砌**：给太多线索读者反而困惑
10. **揭秘太匆忙**：最后1-2章才突然全部解释
11. **死亡太廉价**：受害者要让读者在乎
12. **侦探没有代价**：调查过程不应该毫无风险

### 写作建议
- 先设计好真相，再倒推线索埋设位置
- 写完初稿后从头读一遍，确认线索链完整
- 让你的读者（测试读者）在揭秘前猜凶手，如果太容易猜到就加误导
- 悬疑文的对话最重要——每句对话都可能是线索或误导
