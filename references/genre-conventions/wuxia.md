# 武侠/仙侠 题材套路

## 核心要素
- **江湖**：一个有规则、有恩怨、有门派的社会
- **侠义**：主角的行为准则——路见不平/恩怨分明/义字当先
- **武功**：功法、招式、内力是核心竞争力
- **恩怨**：师门恩情、家族仇恨、兄弟背叛是核心驱动力

## 经典结构
```
少年出山/家变 → 偶遇高人/获得秘籍 → 初闯江湖
→ 结识伙伴/红颜 → 卷入门派纷争 → 发现更大的阴谋
→ 修炼突破/领悟 → 正邪大战 → 归隐/守护
```

## 子类型区分

### 传统武侠
- 古典江湖，拳脚刀剑，内力真气
- 重侠义精神和江湖道义
- 武功体系偏写实（内力/招式/轻功）
- 金庸/古龙/梁羽生风格
- 语言偏古典雅致，重意境

### 仙侠
- 在武侠基础上加入修仙元素
- 飞剑/法术/仙器/渡劫
- 世界观更宏大（凡界→仙界→天界）
- 感情线比重通常较高（仙侠多虐恋）
- 语言偏空灵飘逸

### 新派武侠
- 现代叙事手法讲古代江湖故事
- 解构传统武侠的套路和价值观
- 反英雄/灰色道德更多
- 可以融合悬疑、权谋元素

## 常用冲突模板

| 冲突类型 | 描述 | 写法要点 |
|----------|------|----------|
| 师门灭门 | 师门被灭→主角逃出→查明真相→复仇 | 灭门者要有自洽的理由 |
| 正邪对立 | 正道门派vs魔教 | 正邪界限可以模糊 |
| 武林大会 | 各门派聚集比武/议事 | 利益博弈>单纯比武 |
| 夺宝争锋 | 神兵/秘籍/宝藏引发的争夺 | 宝物要有独特设定和历史 |
| 身世之谜 | 主角的真实身份渐次揭开 | 身世与主线冲突要有关联 |
| 江湖恩怨 | 上一代的恩怨延续到下一代 | 让读者理解仇恨的来龙去脉 |
| 侠义抉择 | 义气vs感情/大义vs私情 | 没有标准答案的选择更有张力 |
| 门派政治 | 掌门继承/联盟分裂/叛徒 | 比"谁打赢谁"更有深度 |

## 武功体系设计

### 武侠体系
| 元素 | 设计要点 |
|------|----------|
| 内力 | 分层级（一流/绝顶/宗师），修炼有上限 |
| 招式 | 每门派有特色招式，忌千人一面 |
| 轻功 | 移动方式的差异化（飘逸/迅猛/诡异） |
| 暗器 | 辅助战斗手段，适合配角使用 |
| 禁忌功法 | 威力大但有代价（走火入魔/折寿） |

### 仙侠体系
| 阶段 | 特征 | 关键转折 |
|------|------|----------|
| 练气 | 引气入体，感知天地 | 寻找灵根/入门 |
| 筑基 | 打下修炼根基 | 可能需要丹药/机缘 |
| 金丹 | 凝聚金丹，战力跃升 | 天劫/心魔 |
| 元婴 | 法力质变，可以飞行 | 进入更大的修仙圈 |
| 化神/渡劫 | 接近仙道 | 天劫考验/飞升抉择 |

### 战斗描写要点
- **武侠战斗**：重招式交锋和细节（剑锋偏了三寸）、心理博弈、以弱胜强的智谋
- **仙侠战斗**：重法术特效和天地异象、元素对抗、法宝碰撞
- 两者共通：战斗中必须有情感/信念层面的碰撞，不只是打打杀杀

## 江湖生态设计

### 门派层级
```
超然势力（隐世高手/远古宗门）
    ↕
一等门派（3-5个）── 控制江湖格局
    ↕
二等门派（5-10个）── 依附一等势力
    ↕
散修/游侠 ── 江湖底层
```

### 江湖规则
- 江湖规矩：比武不伤命/一诺千金/师命不可违
- 打破规矩的代价：被江湖追杀/名声扫地/众叛亲离
- 灰色地带：规矩是否适用于所有人？正道也有小人

## 角色原型

### 主角原型
| 类型 | 特征 | 典型开局 |
|------|------|----------|
| 少年侠客 | 热血正义/初出茅庐 | 下山/家变/被救 |
| 浪子剑客 | 潇洒不羁/身负秘密 | 独行江湖/被卷入事件 |
| 复仇者 | 隐忍冷静/目标明确 | 灭门/背叛/师父被杀 |
| 隐世高手 | 不愿出山/被迫入世 | 太平被打破/旧友求助 |

### 经典配角
| 类型 | 功能 | 常见安排 |
|------|------|----------|
| 师父 | 传道授业/精神导师 | 前期指引→后期揭示身世 |
| 兄弟 | 患难之交/生死相依 | 共同经历→可能分道扬镳 |
| 红颜 | 感情牵绊/成长动力 | 相知相恋→分别→重逢 |
| 大反派 | 终极对手/镜像角色 | 有自己的道→侠道对决 |
| 神秘人 | 线索提供者/暗中观察 | 关键时刻现身→真实身份反转 |

## 氛围与语言

### 武侠的语言风格
- 古典但不古板：可以用现代语序，但词汇偏古
- 诗意但不空洞：景物描写要与情绪/剧情呼应
- 对话简洁有力：侠客不废话，一句话一个态度
- 战斗描写要有节奏：长短句交错，紧张处短句连击

### 意境营造
| 场景 | 意境关键词 | 常用元素 |
|------|------------|----------|
| 江湖夜饮 | 豪迈/洒脱 | 篝火、酒坛、剑舞 |
| 绝壁修炼 | 孤独/超脱 | 悬崖、瀑布、云雾 |
| 门派大会 | 紧张/暗流 | 大殿、位次、暗号 |
| 古战场 | 沧桑/悲壮 | 残剑、枯骨、夕阳 |
| 小镇客栈 | 烟火/偶遇 | 酒旗、说书人、路人 |

## 避坑指南

### 严重问题（必须避免）
1. **武力值崩塌**：前面打不过的对手后面一招秒杀
2. **侠义空谈**：主角喊着侠义实际行为自私
3. **门派千篇一律**：每个门派都差不多没有特色
4. **女角色只是花瓶**：侠女应该有自己的武功和立场
5. **反派脸谱化**：大魔头只是纯粹邪恶没有动机

### 常见问题（注意规避）
6. **修炼枯燥**：闭关三个月一笔带过或流水账
7. **江湖太小**：来来去去就那几个人
8. **时代感缺失**：古人说现代话/行为不符合时代
9. **武功设定矛盾**：前后力量体系不一致
10. **情感线生硬**：打着打着突然就爱上了

### 写作建议
- 武侠的核心不是武，是侠——侠义精神是灵魂
- 每个门派设计2-3个独特标签（武功特色/门规/服饰/行事风格）
- 武打场景不超过全章30%，用情感和智谋填充其余部分
- 仙侠要处理好"修仙"和"人间情感"的矛盾——这往往是最好的戏
