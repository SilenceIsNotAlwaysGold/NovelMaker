# 科幻 题材套路

## 核心要素
- **科技设定**：故事的科技基础要自洽（不需要真实，但要内部逻辑一致）
- **思想实验**：好的科幻是"如果……会怎样"的延伸
- **人文关怀**：科技是背景，人的处境才是核心
- **世界观新颖**：给读者一个从未想象过的世界

## 经典结构
```
日常/现状 → 异变/发现（科技突破/外星接触/灾难降临）
→ 适应新规则 → 探索+冲突 → 真相逐渐浮现
→ 危机升级（科技失控/文明碰撞/生存抉择）
→ 终极抉择 → 新秩序/代价
```

## 子类型区分

### 太空歌剧/星际
- 星际航行、外星文明、舰队战争
- 世界观宏大，适合长篇连载
- 核心爽点：探索未知+文明碰撞+史诗战争
- 注意：科技设定可以软一些，重在故事和格局

### 末世/废土
- 文明崩塌后的生存故事
- 人性的极限考验是核心看点
- 资源争夺/势力博弈/生存vs道德
- 注意：末世规则要早期确立，不能随意改变

### 赛博朋克
- 高科技低生活，人与机械的边界模糊
- 大公司控制一切，个人反抗体制
- 意识上传/义体改造/AI觉醒
- 注意：赛博朋克的核心是反乌托邦，不是装酷

### 硬科幻
- 以真实或合理推演的科学为基础
- 重逻辑推演和技术细节
- 适合思想实验型故事
- 注意：硬核不等于枯燥，技术要服务于故事

### 软科幻/社会科幻
- 科技是背景，探讨的是社会/哲学问题
- 人性、政治、伦理才是核心
- 科技设定可以简化
- 注意：社会议题要有深度，不能流于表面

### 灵气复苏/末世觉醒
- 现代世界突然出现超自然力量
- 融合了科幻和玄幻的元素
- 等级体系+现代社会碰撞
- 注意：要处理好"超能力"和"现代文明规则"的矛盾

## 常用冲突模板

| 冲突类型 | 描述 | 写法要点 |
|----------|------|----------|
| 人vs科技 | AI失控/基因改造后果/科技伦理 | 科技发展的逻辑要合理 |
| 人vs环境 | 末世生存/星球殖民/资源枯竭 | 环境压力要持续且真实 |
| 人vs人 | 在极端环境下的人性博弈 | 每方都有合理立场 |
| 文明碰撞 | 不同文明的接触与冲突 | 避免简单的善恶二分 |
| 身份危机 | 克隆体/AI觉醒/记忆篡改 | "我是谁"的哲学困境 |
| 生存抉择 | 少数人vs多数人/个体vs群体 | 没有标准答案的道德困境 |
| 权力垄断 | 科技被少数人掌控 | 权力腐败的逻辑链 |
| 时间困境 | 时间旅行/平行宇宙带来的悖论 | 时间规则要前后一致 |

## 科技设定原则

### 硬度光谱
| 硬度 | 描述 | 适用场景 |
|------|------|----------|
| 硬科幻 | 基于已知科学推演 | 近未来/技术型故事 |
| 中硬 | 有科学基础但有合理外推 | 太空歌剧/赛博朋克 |
| 软科幻 | 科技是背景不深究原理 | 社会实验/末世求生 |
| 科学幻想 | 魔法般的科技，重在世界观 | 灵气复苏/超能力 |

### 设定的三条铁律
1. **内部自洽**：你的科技规则可以不真实，但不能前后矛盾
2. **有限制**：任何科技都有代价/局限（不能万能）
3. **影响社会**：一项科技会如何改变社会结构？（要想清楚）

### 避免的设定陷阱
- 科技解释太多 → 变成论文，读者跳过
- 科技解释太少 → 变成魔法，失去科幻感
- **最佳实践**：用角色的日常行为展示科技，而非专门开一段解释

## 末世/废土写作要点

### 末世规则设定清单
| 要素 | 需要确定 |
|------|----------|
| 灾难类型 | 什么导致了文明崩塌？ |
| 时间线 | 末世发生多久了？ |
| 残存文明 | 还有哪些社会组织存在？ |
| 资源状况 | 什么是稀缺的？什么是充足的？ |
| 威胁 | 除了人之外还有什么威胁？ |
| 科技残余 | 旧世界的科技还能用多少？ |

### 末世生存的真实感
- 食物/水/药品是硬通货
- 信任是最奢侈的东西
- 道德底线会被不断试探
- 小社群比大组织更有故事性
- 不要忽略心理创伤和PTSD

## 赛博朋克写作要点

### 核心元素清单
| 元素 | 描述 |
|------|------|
| 大公司 | 控制经济/政治/信息的巨型企业 |
| 底层 | 贫民窟/黑市/地下世界 |
| 义体 | 人体改造/机械化/增强 |
| 网络 | 虚拟空间/黑客/数据战争 |
| AI | 人工智能的角色和威胁 |
| 阶层 | 极端贫富差距 |

### 赛博朋克的视觉与氛围
- 霓虹灯+雨夜+狭窄街道
- 全息广告+破旧建筑
- 人与机械的视觉混合
- 信息过载+精神空虚

## 角色原型

### 科幻主角原型
| 类型 | 特征 | 适合子类型 |
|------|------|------------|
| 科学家/工程师 | 技术驱动，理性思维 | 硬科幻/AI相关 |
| 幸存者 | 在极端环境中求生 | 末世/废土 |
| 黑客/边缘人 | 反体制，底层视角 | 赛博朋克 |
| 军人/探险家 | 勇敢无畏，执行任务 | 太空歌剧 |
| 觉醒者 | 发现世界真相的普通人 | 社会科幻/反乌托邦 |
| 异种/AI | 非人视角看世界 | AI觉醒/外星文明 |

### 科幻反派设计
- 科幻反派往往不是"坏人"而是"另一种对的人"
- 公司高管的逻辑：效率>人权
- AI的逻辑：最优解不一定考虑人类情感
- 末世领袖的逻辑：少数人的牺牲换取多数人的生存

## 爽点设计

### 科幻特有的爽点
| 爽点 | 描述 | 频率 |
|------|------|------|
| 科技碾压 | 用高科技解决看似无解的问题 | 每卷2-3次 |
| 真相揭示 | 世界/身份/历史的真相 | 递进释放 |
| 智力碾压 | 用知识和逻辑击败对手 | 每卷1-2次 |
| 绝境求生 | 极端环境下的生存突破 | 关键节点 |
| 文明升级 | 获得新科技/新能力 | 每2-3卷 |
| 宏大叙事 | 星际战争/文明命运 | 高潮部分 |

## 避坑指南

### 严重问题（必须避免）
1. **设定前后矛盾**：科技规则说变就变
2. **机械降神**：危机时突然冒出之前没提过的科技
3. **科技万能**：主角的科技没有任何限制和代价
4. **忽视人文**：纯粹堆砌科技概念没有人物和情感
5. **末世太舒适**：末世环境下角色生活得太轻松

### 常见问题（注意规避）
6. **解释过多**：花太多篇幅解释科技原理
7. **术语堆砌**：大量专业术语让读者一头雾水
8. **社会影响缺失**：有了新科技但社会结构完全没变
9. **外星人太像人类**：外星文明不应该是"换了皮的人类"
10. **反派动机薄弱**：尤其是AI/公司类反派要有逻辑
11. **时间旅行悖论**：涉及时间的故事要想清楚规则
12. **克隆/记忆类故事结局太敷衍**："我是我"不能靠一句话解决

### 写作建议
- 科技设定只解释读者理解剧情必需的部分
- "如果这项科技存在，社会会变成什么样？"——这个问题比科技本身重要
- 科幻的核心冲突往往是道德困境，不是技术难题
- 最好的科幻是读完后让人思考现实世界的问题
