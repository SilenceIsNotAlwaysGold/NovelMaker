# 玄幻/修真 题材套路

## 核心要素
- **力量体系**：境界划分清晰，每个境界有明确的能力提升
- **金手指**：主角必须有独特的外挂（系统/传承/血脉/神器）
- **升级节奏**：定期突破，每次突破带来爽感
- **打脸**：核心爽点，被人看不起→当场碾压

## 经典结构
```
废物开局 → 获得金手指 → 初次逆袭 → 进入更大世界
→ 宗门/学院 → 比赛/任务 → 势力争端 → 探索秘境
→ 大陆级冲突 → 飞升/成神
```

## 子类型区分

### 高武玄幻
- 重战斗描写，力量等级差距明显
- 越级挑战是核心爽点
- 适合"天才vs废物"的冲突框架
- 典型：斗破苍穹、武动乾坤

### 低武仙侠
- 重意境感悟，修炼过程偏向求道
- 情感与成长并重，节奏稍慢
- 氛围偏古典，语言更雅
- 典型：凡人修仙传

### 洪荒流
- 以洪荒神话为背景，角色多为上古大能
- 力量体系宏大，准圣、圣人等极高境界
- 重势力博弈（阐教/截教/人族）
- 需要扎实的神话知识储备

### 系统流
- 主角获得系统面板（任务/奖励/商城）
- 系统提供明确的升级路径和目标感
- 爽感来自系统奖励+意外触发
- 注意：系统不能万能，要有限制条件

## 常用冲突模板

| 冲突类型 | 描述 | 频率 | 写法要点 |
|----------|------|------|----------|
| 天才vs废物 | 天才嘲笑主角→被打脸 | 每卷1-2次 | 先充分铺垫对方的傲慢和周围人的不看好，碾压时才爽 |
| 家族恩怨 | 家族被灭/被逐→复仇 | 主线 | 复仇不能一蹴而就，要有阶段性小胜利 |
| 抢夺资源 | 灵药/秘境/传承争夺 | 频繁 | 资源价值要先铺垫，让读者知道这东西有多珍贵 |
| 退婚打脸 | 被退婚→实力碾压回去 | 经典开局 | 重在对方的悔恨和旁人的震惊反应 |
| 宗门比武 | 排名赛/入门考核 | 每卷1次 | 赛制设计要有悬念，不能每场都碾压 |
| 秘境探索 | 机缘与危险并存的封闭空间 | 每2-3卷 | 要有竞争者、有牺牲、有意外收获 |
| 越阶挑战 | 以低境界击败高境界强敌 | 关键战斗 | 必须给出合理理由（金手指/特殊功法/智谋） |
| 师徒传承 | 高人传授/遗迹得宝 | 每卷1次 | 传承要有代价，不能白拿 |

## 力量体系设计原则

### 必须遵守的规则
1. **等级数量适度**：6-9个大境界，每个境界分3-9个小等级
2. **跨级限制**：明确"几个小等级可以跨越"，避免随意越级
3. **突破条件**：感悟/资源/战斗/机缘，不能只靠打坐
4. **天花板**：要有明确的最高等级和到达条件

### 突破场景写法
```
铺垫（3-5章积累）→ 契机出现（战斗/顿悟/丹药）
→ 突破过程（身体变化+精神升华+力量质变）
→ 突破后展示（新能力+旧敌不再是对手）
→ 周围人反应（震惊/恐惧/嫉妒）
```

### 战斗描写层次
| 层次 | 内容 | 字数占比 |
|------|------|----------|
| 技巧层 | 招式交锋、策略对决 | 40% |
| 意志层 | 信念碰撞、心态博弈 | 25% |
| 感官层 | 天地异象、破坏描写 | 20% |
| 情感层 | 战斗背后的情感驱动 | 15% |

## 爽点分布

### 频率控制
- 每3章至少一次小打脸（口头碾压/小战斗）
- 每卷至少一次大突破（境界提升）
- 每2卷一次换地图（小地方→大世界）
- 获宝要有铺垫，不能凭空掉馅饼
- 每5-8章一次中等爽点（赢比赛/得宝物/救人）

### 爽点节奏模板（单卷15章示例）
```
第1章：新环境受辱→读者期待
第2-3章：隐忍积累+暗中修炼
第4章：小打脸（配角级敌人）★
第5-6章：发现机缘线索+新的挑战
第7章：中等爽点（赢得资格/获宝）★★
第8-9章：更大的敌人出现+压力升级
第10章：受挫/危机（制造低谷）
第11-12章：突破契机+修炼积累
第13章：大突破★★★
第14章：以新实力碾压之前的敌人★★★★
第15章：卷末终极战斗+下卷伏笔★★★★★
```

## 换地图机制

### 为什么要换地图
- 避免同一场景写几十章导致审美疲劳
- 扩大世界观格局，提升故事层次
- 为主角提供新的挑战和成长空间

### 换地图时机
| 触发条件 | 说明 |
|----------|------|
| 实力超越当前环境 | 修为太高，当前地方没有对手 |
| 剧情驱动 | 寻找某样东西/追踪某人 |
| 被迫离开 | 被追杀/被放逐/家族被灭 |
| 机缘引导 | 发现线索指向更大的世界 |

### 换地图流程
```
对当前世界的留恋（告别场景）→ 出发/被迫离开
→ 新世界初见（文化冲击+实力落差）→ 重新定位
→ 在新世界站稳脚跟 → 开始新的挑战
```

## 角色原型

### 主角原型
| 类型 | 特征 | 适合开局 | 注意事项 |
|------|------|----------|----------|
| 废柴逆袭型 | 开局弱小，获得金手指崛起 | 被欺凌/被退婚 | 变强要有过程，不能一步登天 |
| 天才陨落型 | 曾经强大，因故跌落谷底 | 被废修为/遭背叛 | 重建过程比第一次更有故事性 |
| 扮猪吃虎型 | 隐藏实力，关键时刻亮出底牌 | 潜伏入某势力 | 暴露时机要选在最爽的点 |
| 果断决绝型 | 做事利落不拖泥带水 | 复仇开局 | 避免变成无情杀人机器 |

### 常见配角
| 类型 | 功能 | 常见命运 |
|------|------|----------|
| 忠仆/兄弟 | 陪伴+搞笑+承担危险 | 保护主角受伤→激发主角愤怒 |
| 神秘老者 | 传授功法+指引方向 | 关键时刻现身→后期揭示身份 |
| 红颜知己 | 情感线+被抓→救人动力 | 多为治愈系或战斗系 |
| 同门师兄 | 友好→暗藏心机→背叛 | 伏笔型角色 |
| 阶段性反派 | 每卷的BOSS | 有自己的逻辑→不要太脑残 |

## 避坑指南

### 严重问题（必须避免）
1. **无限升级无限换地图**：要有终点，最高境界要在前期暗示
2. **女角色花瓶化**：女性角色要有自己的目标和成长
3. **反派全员脑残**：反派至少给个合理动机，最好有自洽的逻辑
4. **金手指万能**：金手指要有限制条件和使用代价
5. **越级战斗合理性缺失**：每次越级必须有充分理由

### 常见问题（注意规避）
6. **修炼过程枯燥**：不要流水账式写"修炼了三天三夜"
7. **配角工具人**：配角要有自己的性格和行为逻辑
8. **世界观前后矛盾**：力量体系的规则一旦设定不能随意打破
9. **节奏失控**：不能连续5章以上没有爽点
10. **丹药/装备通胀**：宝物的价值要随剧情保持合理
11. **对话模板化**：师父都是"孩子，记住……"，反派都是"蝼蚁！"
12. **情感线突兀**：感情发展需要铺垫，不能突然就爱上了

### 写作建议
- 每次战斗前问自己：这场战斗的**意义**是什么？（不只是打就完了）
- 突破不能太频繁：一卷最多1-2次大突破
- 低谷期是好东西：连续爽会审美疲劳，适当的挫折让后续爽点更爽
- 伏笔在玄幻中至关重要：金手指的来历、师父的身份、血脉的秘密
