# 多线叙事编织技巧

## 一、线路类型

| 类型 | 描述 | 占比建议 |
|------|------|----------|
| **主线** | 核心冲突，全书贯穿 | 50-60% |
| **角色线** | 某角色的个人故事/成长 | 15-25% |
| **感情线** | 爱情/友情/师徒关系发展 | 10-20% |
| **世界线** | 大世界的变化/势力消长 | 5-15% |
| **暗线** | 隐藏的线索，后期揭晓 | 5-10% |

## 二、编织模式

### 模式1：主线穿插
```
主线A → 支线B（1章）→ 主线A → 支线C（1章）→ 主线A
```
主线不停推进，支线穿插其中。适合主线很强的故事。

### 模式2：双线交替
```
A线 → B线 → A线 → B线 → 合流
```
两条线平行推进，最终交汇。适合双主角故事。

### 模式3：散点汇聚
```
A线 ↘
B线 → 汇合点 → 统一主线
C线 ↗
```
多条看似无关的线索最终汇聚。适合悬疑/群像。

## 三、支线管理规则

### 支线的开启
- 每卷同时活跃支线不超过 3 条
- 开启支线必须有明确的收束计划
- 支线必须与主线有关联（哪怕关联要到后面才揭示）

### 支线的维护
- 每条活跃支线至少每 5 章推进一次
- 不要让读者忘记某条支线的存在
- 长时间不推进时，用一句话提醒读者（角色提及、场景暗示）

### 支线的收束
- 支线可以：汇入主线 / 独立收束 / 升级为新主线
- 不允许：悄悄消失不管
- 收束时应产生对主线的影响

## 四、线路切换技巧

### 好的切换时机
- A线达到悬念高点时切到B线（制造期待）
- B线的事件将直接影响A线时切回A线
- 时间/空间上有自然过渡时

### 切换标记
- 空行 + 分隔符（---）
- 明确的时间/地点标记
- 不同的叙述视角

### 切换禁忌
- 不要在无悬念的平淡点切换（读者不会想回来）
- 不要切换后让读者不知道在跟谁（开头几句确认人物和场景）
- 不要一章内切换超过 3 次

## 五、暗线设计

暗线是长篇小说的灵魂——读者回头看时发现"原来第3章就暗示了"。

### 暗线埋设技巧
| 技巧 | 示例 |
|------|------|
| **不经意的细节** | "他注意到角落有个人在看他"（这个人后来很重要） |
| **错误的解读** | 角色A以为事件是意外，读者也以为，但其实是人为 |
| **双关对话** | 角色说的一句话表面意思和真实意思不同 |
| **背景信息** | 新闻/公告/环境描写中藏着线索 |

### 暗线揭示时机
- 不要太早（读者还没忘记就揭示=没有惊喜）
- 不要太晚（读者完全忘记了=没有冲击）
- 最佳：读者隐约觉得不对劲，但说不上来，然后揭示时恍然大悟
