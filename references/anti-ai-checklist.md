# AI痕迹清除检查清单

## 一、黑名单词汇（出现即替换）

### 华丽空洞词
- 璀璨、瑰丽、绚烂、绚丽、斑斓
- 宛如、犹如、恰似、仿若、好似
- 熠熠生辉、流光溢彩、美轮美奂
- 波光粼粼、星光璀璨

### AI高频连接词
- 此外、然而、尽管如此、值得注意的是
- 不可否认、毫无疑问、事实上
- 总而言之、综上所述

### 过度修饰
- 深邃的眼眸、如墨的长发、精致的五官
- 修长的手指、挺拔的身姿
- 温润如玉、清冷如霜

### 抽象情感堆砌
- 心中涌起一股复杂的情感
- 内心深处泛起层层涟漪
- 一种说不清道不明的感觉
- 心里五味杂陈

## 二、模式检测

### 四字成语连用（连续出现2个以上即报警）
```
错误：他心潮澎湃，热血沸腾，斗志昂扬
正确：他的手在抖。不是因为害怕。
```

### "他感到..."模式
```
错误：他感到一阵愤怒涌上心头
正确：他猛地站起来，椅子向后滑出半米
```

### "不禁..."模式
```
错误：她不禁红了眼眶
正确：她别过脸，用袖子快速擦了一下眼角
```

### 排比三连
```
错误：他看到了希望，看到了未来，看到了可能
正确：他看到了一扇门。虚掩着的。
```

## 三、修复策略

### 策略1：抽象→具体动作
| 原文 | 修改 |
|------|------|
| 他很愤怒 | 他一拳砸在桌上，茶杯弹起来摔碎了 |
| 她很伤心 | 她蹲在墙角，指甲掐进掌心 |
| 他很紧张 | 他反复检查了三遍门锁 |

### 策略2：书面→口语化
| 原文 | 修改 |
|------|------|
| "我认为我们应当立即采取行动" | "赶紧走，别磨蹭" |
| "此事非同小可" | "出大事了" |
| "我对此深感遗憾" | "操。" |

### 策略3：长句→长短交错
```
错误（节奏单调）：
他走进了那间昏暗的房间，看到了桌上放着的那把沾着血迹的刀，心中突然涌起了一股难以名状的恐惧。

正确（节奏变化）：
他推开门。房间很暗。
桌上有把刀。刀上有血。
他往后退了一步。
```

### 策略4：心理描写→外化
```
错误：她内心充满了矛盾，一方面想要原谅他，另一方面又无法忘记那个夜晚的背叛。
正确：她把他的照片从钱包里抽出来，看了很久，然后又放了回去。
```

### 策略5：增加角色独特语言
- 给每个角色设定1-2个口头禅
- 给每个角色设定说话时的习惯动作
- 注意不同教育背景的用词差异
- 适当使用方言/俚语/行话

## 四、自查流程

每章写完后依次检查：

1. **全文搜索黑名单词** → 逐个替换
2. **检查四字成语密度** → 每段不超过1个四字成语
3. **检查"他感到/她觉得"出现次数** → 每章不超过3次
4. **检查对话书面感** → 朗读一遍，不像人话就改
5. **检查句长变化** → 连续3句以上长度相同就打散
6. **检查段落首句** → 不能连续3段以"他"或"她"开头
